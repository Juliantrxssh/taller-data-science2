# Taller de Data Science - Diplomado Industria 4.0 📊

¡Hola! Este repositorio contiene los ejercicios, consultas, laboratorios y el desarrollo práctico que estoy realizando como estudiante del **Diplomado en Industria 4.0**. 

El proyecto está enfocado en el aprendizaje y aplicación de herramientas de Data Science, abarcando desde la gestión y estructuración de bases de datos relacionales hasta el análisis exploratorio de datos (EDA) y la manipulación de datasets.

## 🛠️ Tecnologías y Herramientas

* **SQL:** Creación de bases de datos, consultas estructuradas y análisis relacional.
* **Python:** Lenguaje principal para el procesamiento de datos.
* **Jupyter Notebooks (.ipynb):** Entorno interactivo para el desarrollo y visualización del código.
* **Pandas:** Librería fundamental para la manipulación y análisis de datos tabulares.
* **Git & GitHub:** Control de versiones y gestión de ramas (desarrollo principal en la rama `julian`).
